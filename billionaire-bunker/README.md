# Billionaire Bunker Plugin

A Claude Code plugin for billionaire bunker life.

## Installation

Add the marketplace:

```shell
/plugin marketplace add russellslater/claude-plugins
```

Install the plugin:

```shell
/plugin install billionaire-bunker@claude-plugins
```

Or for local development:

```shell
/plugin marketplace add ./claude-plugins
/plugin install billionaire-bunker@claude-plugins
```

## Commands

### `/bunker-status`

Get a fictitious quote from a billionaire about their post-apocalyptic bunker life.

**Usage:**

```shell
/billionaire-bunker:bunker-status
```

Or simply:

```shell
/bunker-status
```
